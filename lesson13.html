1. Introduction to Flexbox

CSS Flexbox exists to give you the technology to create basic and ADVANCED website layouts in flexible and responsive ways.
Flexbox also allows easy RESIZING and REPOSITIONING of flexible containers.

What is a common problem with float design in CSS?

In case when the floating elements are removed from the normal web page flow, they may overlap with other elements on the page. This can cause unexpected space and layout issues, which can lead to web DESIGN BROKENESS

The flex-wrap Property

The CSS flex-wrap property is used to define whether flex objects are forced into a single line or wrapped into MULTIPLE lines.

D) flex-wrap: wrap; (objects will wrap if necessary, default - nowrap)
