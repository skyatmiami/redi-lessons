1. Introduction to CSS Units, font-size

Absolute (fixed) units

Objects for which we specify absolute (or also fixed) values
will have exactly the value that we specify.

  • cm - centimeters
  • mm - millimeters
  • in - inches
  • px - pixels
  • pt - points
  • pc - picas

Relative units

Relative values are useful when we want our objects to change their size when conditions change
(for example, viewing a site on devices with different screen sizes - laptop, phone, tablet).

  • em
  • ex
  • ch
  • rem
  • vw - relative to 1% of the width of the browser window size
  • vh
  • vmin - relative to 1% of the browser's windows smaller dimension
  • vmax
  • %
